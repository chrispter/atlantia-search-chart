## 👨🏻‍💻 Commands

💾 Clone the repository

```bash
git clone https://github.com/chrispter/atlantia-search-chart.git
```

🧶 Install dependencies

```bash
yarn
```
or 

```bash
npm install
```

👷 Generate the assets in development mode

```bash
yarn dev
```
or

```bash
npm run dev
```

🏛️ Generate the assets in production mode

```bash
yarn build
```

or

```bash
npm run build
```
